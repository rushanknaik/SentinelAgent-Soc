# SentinelAgent – Autonomous AI-SOC Platform

An autonomous **AI Security Operations Center (AI-SOC)** built using **LangGraph**, **Llama-3.3-70B**, and **FastAPI** to automate security incident detection, enrichment, and response through a multi-agent architecture.

## 🌐 Live Demo

🔗 **Website:** https://agentsoc-eaq6.onrender.com/

---

## 🚀 Features

- Multi-agent AI workflow powered by LangGraph
- Hybrid rule-based + LLM threat detection
- Threat intelligence integration with VirusTotal, AbuseIPDB, and AlienVault OTX
- MITRE ATT&CK mapping for security incidents
- Prompt injection defenses for secure LLM interactions
- Dockerized FastAPI backend with SQLite caching
- Interactive incident management dashboard
